# Security Policy

## Reporting a Vulnerability

Please report security issues **privately**.

- Email: **furinode@gmail.com**
- Subject: `[guardrail-sdk] Security report`

Include:
- Summary of the issue
- Steps to reproduce
- Impact assessment
- Affected versions/commits (if known)

We will acknowledge receipt and work on a fix ASAP.

## Supported Versions

This project is pre-1.0 and moves quickly. Security fixes will generally target the latest release and `main`.
